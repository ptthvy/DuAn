This website was created to practice applying Cascading Style Sheets (CSS) to an existing HTML structure. All pages share a single stylesheet, which demonstrates how consistent design and layout can be achieved across multiple pages using CSS.

The design focuses on readability, clear navigation, and accessible color contrast. Each element has been styled using semantic HTML tags such as header, navigation, main content, and footer.
