The purpose of this project is to explore how CSS can control the appearance of a website without changing its underlying HTML structure. By using one stylesheet for all pages, the project highlights the efficiency and power of CSS in web design.

Styling choices include background colors, font families, font sizes, spacing, and layout adjustments. Special attention was given to accessibility standards, including sufficient color contrast and readable text spacing.
